jagitor-find

Find git repositories
---------------------

:Manual section: 1
:Date: 2017-06-18
:Author: Jan MatÄ›jka yac@blesmrt.net
:Manual group: jagitor manual

SYNOPSIS
========

  jagitor find <path> [-- [ <expression>... ]]

DESCRIPTION
===========

Find git repositories under <path> and print absolute paths to their root
dirs.

<expression> is an expression of ``man 1 find`` through which you can
further limit the search criteria or perform different actions.

EXAMPLES
========

See ``man 1 jagitor``.
